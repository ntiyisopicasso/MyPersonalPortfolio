# Ntiyiso Picasso - Portfolio

Personal portfolio website showcasing my work as a Web Developer and Computer Science student.

## About

This is the portfolio website of Ntiyiso Junior Manganyi (Picasso), a BSc Mathematics & Computer Science student at the University of Limpopo. The site showcases my skills, projects, and experience in web development and cybersecurity.

## Features

- Modern, responsive design with dark theme
- Smooth animations with Framer Motion
- Real-time GitHub activity integration
- Contact form with email functionality
- Timeline showing education and work experience

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```
