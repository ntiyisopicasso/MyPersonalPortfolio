Getting Started

To run this portfolio locally:

    Clone the repository:

bash
git clone <repo-url>

Navigate to the project directory:

bash
cd portfolio

Install dependencies:

bash
npm install
# or
yarn install

Start the development server:

    bash
    npm run dev
    # or
    yarn dev

    Open your browser and visit http://localhost:3000 (or the port specified) to view the portfolio.

Deployment

The application can be deployed on any static site hosting service such as Vercel, Netlify, or GitHub Pages. Simply build the project with:

bash
npm run build
# or
yarn build

and upload the generated dist folder contents to your hosting platform.
